# Star Charter ✦

A constellation-charting dodge game in a single HTML file. Hold a finger to the
glass and your star drifts toward it; release and it coasts. Connect waypoint
stars into numbered constellation plates while asteroids, comets, and black
holes disagree.

**Play:** https://jordan-doerksen.github.io/star-charter/

No build, no dependencies — `index.html` is the whole game. Part of the Glass
Archive.
